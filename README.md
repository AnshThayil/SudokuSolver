# SudokuSolver

A regursive algorithm to solve the Sudoku Puzzle.

Input should be in it's own text file following this format:
(Use 0s for blank boxes)
```
*********************
0 0 6 * 4 8 1 * 3 0 0
0 2 0 * 0 0 0 * 0 4 0
7 0 0 * 0 0 0 * 0 0 9
*********************
8 0 0 * 0 9 0 * 0 0 4
6 0 0 * 3 4 2 * 0 0 1
5 0 0 * 0 6 0 * 0 0 2
*********************
3 0 0 * 0 0 0 * 0 0 5
0 9 0 * 0 0 0 * 0 7 0
0 0 5 * 7 1 6 * 2 0 0
*********************
```
